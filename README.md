# Javaweb_Project
Create Website for Drink Shop using Java Spring Boot + Thymeleaf + MySQL.
Use case:
1.USERS
+View drinks
+Register
+Login
+Update Information
+Cart
+History 
+Ordering
+Forget Password
+Search
2.EMPLOYEES
+Get orders from users
+View information orders
+History today
3.ADMIN
+Manage user: set authorization, block, infor
+Manage product: CRUD
+Manage category: CRUD
